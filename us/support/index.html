List On Watch

  • Add list : on the main page, tap the "+" button to addd new list
  • Delete List : on the main page, long press any list name, then tap the "-" button following the list name
  • Go to list page : on the main page, tap the name of list
  • Add item in a list : on the list page, tap the "+" button to addd new item
  • Delete item : on the list page, long press any item, then tap the "-" button following the item name
  • Check item : on the list page, tap the unchecked item
  • Unheck item : on the list page, tap the checked item
